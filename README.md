# **🎁 Project Build Guide**

## **IDE**

```
FrontEnd: VSCode
BackEnd: IntelliJ
```

## **저장소 클론하기**

```
깃 저장소를 로컬에 복사합니다.
git clone https://github.com/nyong-nyong/pangparty-fe.git
git clone https://github.com/nyong-nyong/pangparty-be.git
```
</br>

## **Build**

**React FrontEnd**
```
npm i
npm run build
```
**SpringBoot BackEnd**
```
./gradlew clean build --exclude-task test
```
